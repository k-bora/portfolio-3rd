# portfolio-3rd-vue

아래의 기반으로 포트폴리오 제작
- vue.js 사용
- HTML, CSS 기반으로 웹사이트의 기본 레이아웃 설계하고, 웹 표준 및 웹 접근성을 준수하여 작업
- css 전처리기 (sass) 사용
- gsap 이용하여 패럴랙스 효과 사용
- github 파일을 관리
- BEM 네이밍 사용

## 완성작 보기 
미리보기 : https://kimbora-portfolio.netlify.app/

## 프로젝트 실행
- vue를 설치합니다. `npm install`
- vue를 실행시킵니다. `npm run dev`
